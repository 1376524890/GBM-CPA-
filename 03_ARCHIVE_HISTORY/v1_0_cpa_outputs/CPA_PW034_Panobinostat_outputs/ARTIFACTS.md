# Local Artifacts

Large CPA outputs are intentionally not tracked by Git. They remain available in this working directory and are ignored by `.gitignore` to avoid upload failures.

| Local path | Size observed | Reason ignored |
|---|---:|---|
| `../GBM_CPA_PW034_Panobinostat_pred.h5ad` | 275M | Predicted AnnData matrix |
| `../GBM_Universal_Perturbation_Ready.h5ad` | 214M | Prepared input AnnData |
| `../GBM_CPA_model/model.pt` | 33M | Trained model weights |
| `../GBM_CPA_training.log` | 342K | Runtime log, regenerated by training |
| `../GBM_CPA_model/history.csv` | 18K | Runtime training history |
| `../GBM_CPA_model/epoch_history.tsv` | 18K | Runtime training history |

Tracked files in this directory provide the experiment summary and reproducibility notes. Re-run `../scripts/train_cpa_ood.py` to regenerate the ignored artifacts when the input dataset is available.
